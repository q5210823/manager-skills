# v0.1.0 Release Notes

## 中文

`v0.1.0` 是 `经纪人.skills` 的首个公开版本。

这个版本聚焦一件事：

`验证“经纪人角色 + 执行推进”是否比普通待办更有推动力。`

### 本版本包含

1. 基于 AgentSkills 的 `create-manager` 主入口
2. 固定 5 项输入的经纪人生成链路
3. 经纪人人设 + 执行系统双层结构
4. 最小 CLI 入口：`tools/create_manager.py`
5. 本地网页表单入口：`tools/manager_web.py`
6. 示例 manager 与示例输入文件
7. 更新与版本管理基础能力

### 当前定位

这还不是一个完整的效率平台，也不是日历产品。

它当前更像一个：

1. 会拆目标的经纪人
2. 会追进度的督导者
3. 会做复盘的推进系统

### 下一步方向

1. 继续增强 prompt 生成质量
2. 把网页入口扩成更完整的可视化界面
3. 评估接入日历或 Chrome 插件

## English

`v0.1.0` is the first public release of `manager.skills`.

This release focuses on one core question:

`Can a manager-style role plus execution pressure outperform a normal to-do workflow?`

### Included in this release

1. `create-manager` as the AgentSkills entry point
2. a fixed 5-input manager generation flow
3. dual-layer output: execution system + persona
4. minimal CLI entry: `tools/create_manager.py`
5. local browser form: `tools/manager_web.py`
6. example manager and sample input
7. basic update and version management support

### Current product position

This is not yet a full productivity platform or calendar product.

At this stage, it is better understood as:

1. a manager that breaks down goals
2. a role that pushes progress
3. a system that enforces review and correction

### Next directions

1. improve prompt quality
2. expand the local web UI
3. evaluate calendar or Chrome extension integration
